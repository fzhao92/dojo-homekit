Accessories
  • HomeKit Accessory Simulator contains one Bluetooth LE accessory, but can have multiple IP accessories added. The Bluetooth LE accessory can not be deleted or exported.
  • To add an IP accessory to HomeKit Accessory Simulator, select the "+" located in the lower-left corner of the app and select "New Accessory..."; use the keyboard shortcut âŒ˜N; or from the File Menu, select "Accessory..." under the "New" menu item. Enter the accessory information when prompted and select the "Finish" button.
  • To add a bridge to HomeKit Accessory Simulator, select the "+" located in the lower-left corner of the app and select "New Bridge..."; or from the File Menu, select "Bridge..." under the "New" menu item. Enter the bridge information when prompted and select the "Finish" button.
  • To add a bridged accessory to a bridge, first select a bridge from the accessory list on the left side of the window. Then select the "+" located in the lower-left corner of the app and select "New Bridged Accessory..."; or from the File Menu, select "Bridged Accessory..." under the "New" menu item. Enter the bridged accessory information when prompted and select the "Finish" button.
  • Accessories may be deleted by selecting them from the accessory list, and then either pressing the delete key or selecting "Delete" from the "Edit" menu. They can also be deleted by selecting them and clicking "-" in lower-right corner of accessory list. A Bluetooth LE accessory can't be deleted.
  • Please note that the Bluetooth LE accessory does not support becoming a bridge.
  • Names must be unique. If a newly created accessory, bridge, or bridged accessory name is already being used, it will automatically be renamed.
