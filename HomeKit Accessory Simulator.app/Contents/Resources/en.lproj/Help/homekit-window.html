The HomeKit Accessory Simulator Window
  • The HomeKit Accessory Simulator window is divided into two sections.
    • The left side contains the accessory list. The list contains exactly one Bluetooth LE accessory which may not be removed. To this list, IP accessories (which includes regular accessories, bridges, and bridged accessories) may be added or removed.
    • The right side of the window contains the main view and gives detailed information for the selected item.
  • The accessory list displays the accessory name, and if the accessory is identifying itself. The BLE accessory, IP accessories, and IP bridges have an on/off state.
    • There is a hollow dot when the accessory is off.
    • When the accessory is identifying, the blue dot will blink for as long as the accessory is identifying.
  • The bottom of the accessory list contains a "+" button on the left and a "-" button on the right. The "+" creates accessories, bridges, and bridged accessories; and the "-" button deletes them.
  • When selecting an accessory or bridge, the top area of the main view displays the Setup Code, Category, and the number of paired devices. This information does not exist for bridged accessories, as they are paired via their bridge.
  • When selecting any item, the top area of the main view shows characteristics included in the accessory's Accessory Information Service. This top area's upper-right corner can be clicked to collapse or expand the area.
  • The bottom of the main view contains additional services for the accessory.
