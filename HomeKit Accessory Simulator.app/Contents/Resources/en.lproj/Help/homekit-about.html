About HomeKit Accessory Simulator
  • HomeKit Accessory Simulator is an app that communicates with an iOS device the same way an actual HomeKit Accessory would, so that developers can test functionality while developing iOS apps that use the HomeKit API.
  • HomeKit Accessory Simulator supports creating accessories with services, and adding characteristics to those services. HomeKit Accessory Simulator also supports pairing with a device, changing values, and receiving changed values. HomeKit API uses HomeKit Accessory Protocol (HAP) to communicate with the accessories.
  • HomeKit Accessory Simulator can simulate 4 types of accessories:
    • Bluetooth LE Accessories, accessories that talk over Bluetooth LE
    • IP Accessories, accessories that talk over Internet Protocol(IP)
    • Bridge Accessories, accessories that act as an interface between HAP and accessories talking over other transport protocols like ZigBee or Z-Wave.
    • Bridged Accessories, accessories communicating with HomeKit API using Bridge accessories.
